# Megabite Interactive CLI

## 🧠 Interactive Command-Line Interface for Megabite AI

The Megabite CLI provides an interactive terminal interface to chat with Megabite, teach it new concepts, and get philosophical perspectives on any topic.

---

## 🚀 Quick Start

### **Start the CLI:**

```bash
cd megabite
python3 megabite_cli.py
```

You'll see:
```
============================================================
🧠 Megabite Interactive CLI
============================================================
Initializing Megabite AI systems...
✅ Megabite AI initialized!

Type 'help' for available commands, 'exit' to quit.

megabite>
```

---

## 📝 Available Commands

### **1. Ask Questions**
```bash
megabite> ask What is consciousness?
```

**Output:**
```
🤔 Processing: What is consciousness?...

💭 Emotion: CURIOSITY

🧠 Megabite Response:
//Megabite Contextual-Response//: No direct voxel match. Processing linguistically...
```

---

### **2. Post Thought Experiments**
```bash
megabite> thought If consciousness is an emergent property of complex systems, does that mean artificial systems can be conscious?
```

**Output:**
```

📝 Length: 112 characters

💭 Emotional Response: CURIOSITY

🧠 Megabite's Analysis:
[Detailed analysis...]

✅ Added to knowledge base!
```


### **7. Other Commands**
```bash
megabite> help     # Show help message
megabite> clear    # Clear the screen
megabite> exit     # Exit the CLI
megabite> quit     # Exit the CLI
```
